Predicting Power Outages

DSC 80

This was my final project in my DSC 80 course which challenged students to create their own data science project from start to finish. I was given only an uncleaned data set and two weeks to make it my own. In this project I took the information of power outages in the United States and built a prediction model to classify the severity of future power outages.

  • Removed 30% of unsightful data using data cleaning and exploratory data analysis
  • Devised DecisionTreeClassifiers and RandomForestClassifiers to create models with 75% testing accuracy and 95% training accuracy
  • Utilized GridSearch and parity analysis to determine optimal parameters and presence of model biases
